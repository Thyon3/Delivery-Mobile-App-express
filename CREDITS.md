# Credits

## Core Technologies

This project is built with and depends on the following excellent open-source projects:

### Backend Framework
- [Express.js](https://expressjs.com/) - Fast, unopinionated web framework
- [Node.js](https://nodejs.org/) - JavaScript runtime

### Database & ORM
- [PostgreSQL](https://www.postgresql.org/) - Advanced open-source database
- [PostGIS](https://postgis.net/) - Spatial database extender for PostgreSQL
- [Prisma](https://www.prisma.io/) - Next-generation ORM

### Real-time Communication
- [Socket.io](https://socket.io/) - Real-time bidirectional event-based communication

### Caching & Jobs
- [Redis](https://redis.io/) - In-memory data structure store
- [BullMQ](https://docs.bullmq.io/) - Premium message queue

### Payment Processing
- [Stripe](https://stripe.com/) - Payment infrastructure

### Security & Authentication
- [Argon2](https://github.com/ranisalt/node-argon2) - Password hashing
- [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken) - JWT implementation
- [Helmet](https://helmetjs.github.io/) - Security middleware

### Validation & Types
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript
- [Zod](https://zod.dev/) - TypeScript-first schema validation

### Development Tools
- [ESLint](https://eslint.org/) - Linting utility
- [Prettier](https://prettier.io/) - Code formatter
- [Jest](https://jestjs.io/) - Testing framework
- [Nodemon](https://nodemon.io/) - Development server

### Documentation
- [Swagger](https://swagger.io/) - API documentation

### Logging
- [Winston](https://github.com/winstonjs/winston) - Logging library
